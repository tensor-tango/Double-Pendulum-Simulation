Double Pendulum Simulation

Real-time simulation of a chaotic double pendulum system, written in MATLAB and Python.
More detailed scientific- and enginerring-based information of the system, numerical algorithm, motion equations and total summary can be found in dedicated file.

Below is demonstrated Python version of simulation
![Pyhon-Simulation](animation.gif)

Below is demonstrated MATLAB version of simulation
![MATLAB-Simulation](MATLAB_anim.gif)

Main features:
- RK4 (Runge-Kutta) numerical integrator
- Live-time calculation of the motion's equations
- Live-time simulation rendering 
- Total system's energy analysis and live vizualisation
- Phase portrait 
- Tracking of chaotic trajectory of the second mass 

!Importatnt notice!
The masses of both of the lengths are omitted and the whole system is in 2D due to pendulum simplification
